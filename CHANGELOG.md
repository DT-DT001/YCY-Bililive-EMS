# 更新日志

## v1.1.3 - 2026-06-14

### 事件与计算逻辑

- 点赞事件改为优先读取 B 站实际推送的 `click_count`，并补充礼物、点赞消息中的舰长等级字段兼容。
- 同一笔大航海交易同时收到 `GUARD_BUY` 和 `USER_TOAST_MSG` 时只触发一次，避免强度与时间重复累加。
- 强度在配置、调度及设备协议层统一限制为 `0-276`，小数强度使用四舍五入映射到协议整数。
- 拒绝非有限数值、无效通道和重复规则 ID，防止异常配置形成永久输出或重复增幅。
- 重复事件继续只添加增幅时间，不重复加入基础时间；每条事件独立应用自己的强度和时间上限。
- 调度器改用完整实际经过时间扣减，避免蓝牙或界面回调阻塞导致输出时间被延长。
- 顺序播放的新事件始终从用户选择的第一个波形开始。

### 设备与协议

- 修复断线期间输出状态被错误缓存，导致设备重连后相同输出不再下发的问题。
- 重连一代设备时清除旧报文缓存、待发送波形点和后台写任务，避免旧状态写入新连接。
- 一代 A/B 参数完全相同时，包括同时停止，统一使用协议规定的 `0x03` 同步报文。
- 二代设备去重完全相同的控制报文，降低无效蓝牙写入。
- 设备连接期间禁止切换产品代际，防止一代、二代报文混发。

### 界面与发布

- 基础强度和强度上限增加 `0-276` 范围标注。
- Windows 构建明确排除未使用的 PySide6/QtWebEngine，发布压缩包由约 187 MB 恢复至约 29 MB。
- 发布包继续使用 Python 3.12 构建，并且不包含本机用户配置。

## v1.1.2 - 2026-06-13

### 启动兼容性

- 修复部分电脑加载 `Python.Runtime.dll` 失败后程序直接崩溃的问题。
- 内置窗口因 .NET Framework、WebView2 或 pythonnet 环境不兼容而无法启动时，自动打开浏览器兼容模式，直播监听和设备控制功能保持可用。
- 支持设置环境变量 `YCY_BROWSER_MODE=1` 手动使用浏览器兼容模式。
- Windows 发布构建固定使用 Python 3.12、pywebview 6.2.1、pythonnet 3.0.5 和 PyInstaller 6.20.0，避免较新 Python 构建链在不同系统上的兼容性差异。

## v1.1.1 - 2026-06-13

### 一代设备协议

- 修正自定义模式映射：事件强度在波形播放期间保持不变，波形只动态改变频率和脉冲时间。
- 自定义波形完整支持 `1-100Hz` 频率和 `0-100us` 脉冲时间。
- 修正 A、B 同步控制：同一刷新周期且参数完全相同时只发送一条通道号 `0x03` 的 AB 同步报文，避免重复控制。
- A、B 参数不同时继续使用独立通道报文，单独停止一个通道不会影响另一个通道。
- 修正 AB 同步后的通道状态缓存，保证后续分离控制及再次同步不会被旧状态错误去重。
- 一代实时控制使用设备级单一发送时钟，任意两条报文至少间隔 120ms，等待期间仅保留各通道最新波形点。

### 稳定性

- 一代错误码 4 提示增加最近控制报文的通道、模式、强度、频率和脉宽，便于定位固件拒绝的参数。
- 修复 Windows BLE 扫描结果失效后按旧地址连接导致 `Device was not found` 的问题；连接前会重新发现设备并刷新 `BLEDevice` 对象。
- 错误恢复不再重放过期的实时波形点，避免恢复过程破坏当前波形时序。
- 增加协议报文、AB 同步/分离/再次同步、强度恒定及发送间隔测试。
- Windows 发布包增加中文手册 `USER_GUIDE.zh-CN.md`，详细说明全部界面、配置参数、调度逻辑、使用步骤、故障排查和安全事项。

## v1.1.0 - 2026-06-13

### 新增

- 使用内置 WebView2 窗口运行，关闭窗口时同步退出程序并断开设备。
- 增加进入、关注和分享直播间事件。
- 增加配置 JSON 导入与导出，设备和通道绑定不会写入配置文件。
- 支持郊狼 V2、V3 以及 `Dungeonlab+pulse` 多小节波形。
- 内置 12 个波形改用设备协议固定模式报文。
- 主界面区分固定模式和自定义实时波形显示。

### 优化

- 改用 `bilibili-api-python` 监听直播间，并完善心跳超时自动重连。
- 修正事件抢占、独立强度上限、时间上限、重复触发增幅和总剩余时间。
- 修正顺序、随机播放及波形重新开始逻辑，随机模式不连续重复同一波形。
- 优化实时波形滚动、停止后归零和幅值稳定性。
- 一代设备控制报文限制为至少 100ms 间隔。
- 一代停止和静音报文使用协议有效参数，降低错误码 4 出现概率。
- 程序启动时自动选择可用本地端口，避免端口占用导致无法打开。

### 修复

- 修复停止监听后输出未结束、事件记录未清除的问题。
- 修复剩余时间显示偶发跳变。
- 修复一代错误恢复重放过期实时波形点的问题。
- 修复新打包程序携带开发配置的问题。
- 修复配置页字号、复选框尺寸抖动和窗口默认位置问题。

### 已知问题

- 郊狼设备与役次元一代设备的电刺激参数模型不同，导入波形属于近似转换，体感无法保证完全一致。
- B站目前不提供可靠的单用户离开和取关实时通知，对应规则仅保留配置与模拟能力。
